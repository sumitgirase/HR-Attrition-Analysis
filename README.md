
# 📊 HR Attrition Analysis Dashboard

A complete end-to-end Data Analytics project that analyzes employee attrition using Python, SQL, Excel, and Power BI. The project focuses on discovering the key factors contributing to employee turnover and presenting actionable insights through an interactive dashboard.

---

## 📌 Project Overview

Employee attrition is one of the biggest challenges faced by organizations. High attrition leads to increased recruitment costs, productivity loss, and reduced employee morale.

This project analyzes HR data to identify trends, patterns, and factors influencing employee attrition. The final outcome is an interactive Power BI dashboard that helps HR professionals make data-driven decisions to improve employee retention.

---

## 🎯 Objectives

- Clean and preprocess HR employee data
- Perform Exploratory Data Analysis (EDA)
- Analyze employee attrition trends
- Execute SQL queries to answer business questions
- Build an interactive Power BI dashboard
- Generate actionable business insights

## 💡 Key Business Insights

- Sales department showed higher employee attrition.
- Employees with lower monthly income were more likely to leave.
- Employees with fewer years at the company had higher attrition.
- Certain job roles experienced significantly higher turnover.
- Attrition varied across age groups and education fields.
